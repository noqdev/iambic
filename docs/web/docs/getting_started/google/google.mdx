In this tutorial, you will configure IAMbic to manage Google Groups.
By the end of this tutorial, you will have performed the following steps:

1. Create a basic configuration for IAMbic
2. Import existing Google Groups into your IAMbic templates repository
3. Create a Google Group with multiple users
4. Provide temporary (break-glass) access to your groups
